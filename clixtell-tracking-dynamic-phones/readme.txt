Clixtell Tracking & Dynamic Phones integrates Clixtell click fraud detection and dynamic phone number insertion into your WordPress site.

== Description ==

**Clixtell Tracking & Dynamic Phones** helps businesses protect their advertising budget and improve conversion tracking by integrating Clixtell’s advanced click fraud detection and dynamic call tracking technology into WordPress.

With this plugin you can:

* Detect and block fraudulent clicks
* Track phone calls accurately from paid traffic
* Enable Dynamic Phone Insertion (DNI)
* Integrate seamlessly with your existing Clixtell account
* Avoid complex code changes or manual script insertion

An active **Clixtell account** is required to use this plugin.

Learn more at [https://www.clixtell.com](https://www.clixtell.com)

== Features ==

* Easy WordPress integration
* Dynamic Phone Insertion (optional toggle)
* Automatic script loading
* Clean and secure WordPress Settings API usage
* Lightweight and performance-friendly
* No theme modification required

== Installation ==

1. Upload the `clixtell-tracking` folder to the `/wp-content/plugins/` directory  
   **OR**
2. Install the plugin through the WordPress Plugins screen
3. Activate the plugin through the **Plugins** menu in WordPress
4. Navigate to **Settings → Clixtell**
5. Enable *Dynamic Call Tracking* if desired
6. Log in to your Clixtell dashboard to complete setup

== Configuration ==

1. Go to **Settings → Clixtell**
2. Check **Activate Dynamic Call Tracking** to enable Dynamic Phone Insertion
3. Save changes
4. Ensure your Clixtell account is properly configured

Tracking scripts are automatically injected on the frontend once enabled.

== Frequently Asked Questions ==

= Do I need a Clixtell account? =
Yes. This plugin requires an active Clixtell account to function.

= Does this plugin slow down my site? =
No. Scripts are lightweight and loaded efficiently using WordPress enqueue methods.

= Can I enable or disable Dynamic Phone Tracking? =
Yes. You can toggle Dynamic Phone Tracking from the plugin settings page.

= Does this work with all themes? =
Yes. The plugin is theme-agnostic and works with all properly coded WordPress themes.

= Is this compatible with caching plugins? =
Yes. Clixtell scripts are compatible with most caching and optimization plugins.

= Does the plugin collect personal data? =
The plugin itself does not store personal data. All tracking and analytics are handled by Clixtell according to their privacy policy.
